About the project

From the structure of this site — catalog pages, book detail pages, purchase confirmation pages, user login/register, and an admin panel — it's clear that "Book Haven" is designed as a full-featured online bookstore. Features include a searchable catalog, detailed book pages, user accounts with cart and checkout flow, secure payments, and an admin interface to manage inventory and orders.
